## 1. Geographic Area
This map covers the **City of Seattle, Washington**, focusing on the distribution and boundaries of city-managed parklands and open spaces.

## 2. Tile Set Details
Below are the details for the thematic tile set generated for this project:

| Tile Set Name | Zoom Levels | Description |
| :--- | :--- | :--- |
| **Seattle Parks Thematic Layer** | 10 - 15 | A thematic visualization of Seattle parks, categorized by their owner/name using a qualitative color scheme. |
